# Antigravity - Personal Stylist Template

A premium, luxury HTML template tailored for Personal Stylists, Image Consultants, and Fashion Professionals.

## Features

- **Premium UI/UX:** Glassmorphism, editorial typography, and minimalist luxury aesthetic.
- **Dark/Light Mode:** Fully supported with a persistent theme toggle.
- **Responsive Design:** Built on Bootstrap 5 Grid for perfect mobile adaptation.
- **Smooth Animations:** Powered by GSAP and custom CSS keyframes.
- **Pages Included:**
  - Home (v1 & v2)
  - Services & Service Details
  - Portfolio
  - About Us
  - Journal (Blog) & Details
  - Pricing
  - Contact
  - Authentication (Login/Signup)
  - 404 Error

## Tech Stack

- **HTML5**
- **CSS3** (Custom Properties, Flexbox, Grid)
- **Bootstrap 5** (Grid Only)
- **Vanilla JavaScript** (ES6+)
- **GSAP** (GreenSock Animation Platform)

## Setup

1. Open `index.html` in your browser to view the template.
2. Customize the content in the HTML files.
3. Replace image placeholders (Unsplash URLs) with your local assets in `assets/images/` if desired.

## Customization

- **Colors:** Edit `:root` variables in `assets/css/style.css` and `assets/css/dark.css`.
- **Fonts:** Currently using `Playfair Display` (Headings) and `Inter` (Body).
- **Animations:** Tweaks can be made in `assets/js/animations.js`.
